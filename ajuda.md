# Ajuda - Projeto Web PetShop Gabriel Souza

## Sobre o projeto
Este projeto foi desenvolvido para a Fase 1 da disciplina de Fundamentos de Sistemas Web.

O objetivo é construir a estrutura inicial de um sistema web para um petshop utilizando apenas HTML, sem customizações com CSS ou Bootstrap e sem implementação de funcionalidades com JavaScript.

## Estrutura da página
A página foi organizada nas seguintes partes:

### 1. Cabeçalho
O cabeçalho apresenta:
- nome do petshop
- descrição inicial
- menu de navegação interna

### 2. Corpo da página
O corpo do site apresenta:
- uma seção de apresentação
- três categorias de produtos:
  - produtos para cães
  - produtos para gatos
  - acessórios para pets
- dois produtos em cada categoria, contendo:
  - imagem
  - descrição
  - valor
- uma seção de serviços contendo:
  - descrição
  - valor
  - informação sobre tele-busca
- um espaço para pontuar os diferenciais do petshop

### 3. Rodapé
O rodapé contém:
- dados de contato
- links de navegação
- informações de autoria

## Tecnologias utilizadas
- HTML5

## Observações
- O projeto foi desenvolvido somente com HTML.
